# Die häufigsten Fehler bei wissenschaftlichen Abschlussarbeiten

- We just build a system.... Here is how we did it....
  - Was hier grundsätzlich fehlt sind Informationen zur Analyse der wesentlichen technischen, fachlichen, funktionalen und nicht-funktionalen Anforderungen an das System?
  - Welche Aspekte müssen in welcher Ausprägung erfüllt sein, damit das System oder ein Teilaspekt davon als erfolgreich im Sinne der Zweckmäßigkeit angesehen werden kann?
  - Auf welcher Basis und mittels welcher Methodiken wurden die Anforderungen erarbeitet?
  - Wie wurden die Anforderungen verifiziert und validiert?
  - Gab' es qualitätssichernde Maßnahmen? Falls ja, welche?
