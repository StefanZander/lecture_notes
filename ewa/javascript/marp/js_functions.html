- $\leadsto$ "you can do with functions anything you can do with objects" { .kursiv}

- Function declaration are **hoisted** to the top of the context^1^ - ie., the function name is known ahead of time - Hence, such functions can be used before they are defined

**Example**

```js const x = (x, y) => { return x * y }; ```
